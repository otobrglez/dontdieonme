### dontdieonme

"Don't die on me is" simple Ruby app that makes request to website every 5 minues.

Make sure you add following to environment variables:
 - URL_TO_GET
 - ROLLBAR_ACCESS_TOKEN
 - ROLLBAR_ENDPOINT
